Programação Web com Django

Esta cadeira oferece uma introdução abrangente ao desenvolvimento de aplicações web utilizando o framework Django, com foco especial na plataforma PythonAnywhere. Durante o curso, os alunos são guiados desde os conceitos fundamentais de programação em Python até a construção de aplicações web completas e funcionais.

Funcionamento da Cadeira

O curso começa com uma revisão das bases de Python, abordando conceitos como sintaxe, estruturas de dados e funções. Em seguida, os alunos são introduzidos ao framework Django e sua arquitetura MVT (Model-View-Template), aprendendo como estruturar e organizar projetos web de forma eficiente.

Os principais tópicos cobertos incluem a criação de modelos para representar dados no banco de dados, o uso de formulários para coletar informações dos usuários, a implementação de autenticação de usuários, a manipulação de URLs e views para controlar o fluxo da aplicação, e a utilização de templates para renderizar páginas HTML dinamicamente.

Além disso, os alunos têm a oportunidade de explorar temas avançados, como a criação de APIs para interação com outras aplicações, a integração de CSS para estilizar as páginas web, e o uso de JavaScript para adicionar interatividade e dinamismo aos projetos.

Linguagens Utilizadas

  • Python
  • CSS
  • JavaScript
  • HTML
